Instructions :

F1 : passer les parties de texte

Left-Click : Pour faire bouger de case le joueur

Space: Pour faire l'attaque du joueur

enter : fini ton tour (le premier enter pour finir le tour bouger, le deuxieme pour finir le tour attaque)

Pour le menu, il faut cliquer sur le carré qui tourne.

Le joueur doit détruire le néant (les tuiles noirs qui s'ajoute a chaque tour) pour gagner. Si le joueur
se fait entourer par le néant, alors il perd.

Description du jeu : c'est un combat entre l'infit et le néant. Le néant veut prendre du terrain, tandis que 
l'infini veut l'arrêter.

Styles : Jeu de rôle, Tour par tour

Thème : Infini et néant
